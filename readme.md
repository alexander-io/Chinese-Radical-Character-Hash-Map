# Chinese Radical Character Hash Map
## 中文激进地图

## Usage linux/unix

```bash
# ensure that nodejs javascript runtime is installed
git clone https://github.com/alexander-io/Chinese-Radical-Character-Hash-Map.git
cd Chinese-Radical-Character-Hash-Map/
node x.js
# input : ...
# input : turtle
# key :  turtle -> { number: '213',
#   character: '龜',
#   pinyin: 'guī',
#   english: 'turtle' }

# input : field
# key :  field -> { number: '102',
#   character: '田',
#   pinyin: 'tián',
#   english: 'field' }

# input : moon
# key :  moon -> { number: '74', character: '月', pinyin: 'yuè', english: 'moon' }

# radicals.json will be output
```
